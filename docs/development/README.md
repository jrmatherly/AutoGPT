## Developer Tools Overview

### Drift
Codebase intelligence tool that helps AI agents understand your code patterns and conventions.

### Mise
Unified tool version manager and task runner for Python, Node.js, and development tasks. Replaces Make, asdf, nvm, pyenv, etc.

### Better Icons
Icon search and integration tool supporting 200,000+ icons from 150+ collections. Available as MCP server or CLI.
