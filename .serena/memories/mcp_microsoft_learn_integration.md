## Quick Reference

| Tool | Use For |
|------|---------|
| `microsoft_docs_search` | Quick concept lookups, API references |
| `microsoft_code_sample_search` | Working code examples (add `language` filter) |
| `microsoft_docs_fetch` | Complete tutorials, deep dives |

## Coverage

- **Excellent**: TypeScript, Playwright, GitHub Actions, VS Code
- **Partial**: Next.js, React (use Vercel/React docs as primary)
- **None**: Pydantic, FastAPI, Prisma (use their official docs)

## When to Use

- ✅ TypeScript type system questions
- ✅ Playwright testing patterns
- ✅ GitHub Actions workflows
- ❌ AutoGPT-specific code (use Serena/Grep)
- ❌ Third-party libraries (use Context7/web search)
